# 🌦️ Simple Weather App

A **Simple Weather Application** built using **HTML, CSS, and JavaScript** that allows users to search for any city and get **real-time weather information**.

The application fetches weather data from a **Weather API** and dynamically updates the UI using JavaScript.

![Weather App Preview](./weather-app-preview)

## 🚀 Features

- 🌍 Search weather by **city name**
- 🌡️ Displays **temperature in °C**
- ☁️ Shows **weather condition**
- 📅 Displays **current date**
- ⚡ Real-time weather data using **API**
- 📱 Responsive and clean UI

---

## 🛠️ Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript**
- **Weather API**
