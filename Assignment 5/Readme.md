
# 🗒️ Sticky Notes Web App

A simple **Sticky Notes Web Application** built using **HTML, CSS, and JavaScript**.

This project allows users to quickly create digital notes on a web page, similar to physical sticky notes used for reminders or quick tasks. Sticky note apps help users organize ideas and tasks in a simple visual way. :contentReference[oaicite:0]{index=0}

# 🚀 Features

- Create new sticky notes
- Write and edit notes easily
- Simple and clean UI
- Dynamic note creation using JavaScript
- Beginner-friendly DOM manipulation project

---

# 🛠️ Technologies Used

- **HTML5** – Structure of the webpage  
- **CSS3** – Styling the sticky notes and layout  
- **JavaScript (Vanilla JS)** – Creating notes dynamically and handling user interaction  

Sticky note apps commonly use **DOM manipulation and browser events** to create and manage notes on a page. :contentReference[oaicite:1]{index=1}
